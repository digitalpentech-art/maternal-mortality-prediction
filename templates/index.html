About This System

Healthcare AI for Maternal Safety

Developed for Maiduguri, Borno State, this system uses advanced machine learning to identify high-risk pregnancies early, enabling clinicians to take preventive action.

Technical Stack
  • Backend: Flask REST API, Scikit-learn, TensorFlow, SHAP
  • Frontend: Bootstrap 5, Plotly.js, JavaScript (Fetch API)
  • Models: Random Forest Classifier, Multilayer Perceptron (ANN)
Medical Disclaimer: This AI system is an assistive behavioral risk assessment tool designed for research and investigative support. It is not a clinical diagnosis and should not be used as sole evidence in medical decision-making.
